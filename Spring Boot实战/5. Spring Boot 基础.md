## 5. Spring Boot 基础

 ### 5-1 Spring Boot 概述

- 习惯优于配置

- 独立运行，内嵌servlet容器
- 准生产的应用监控
- 优点
  - 快速构建
  - 对主流开发框架的无配置集成
  - 项目独立可运行，无需外部的servlet容器
  - 提供运行时的应用监控
  - 极大地提高了开发，部署效率
- spring-boot-devtools 来进行开发热部署

### 5-2 Spring Boot 快速搭建

- spring-boot-starter-parent 常用的包可以省去version标签
- @SpringBootApplication 主要目的是开启自动配置
